    ontopia-topicmaps - Query topic maps with Ontopia¶ ↑

    VERSION¶ ↑

    This documentation refers to ontopia-topicmaps version 0.0.2

    DESCRIPTION¶ ↑

    Query topic maps with Ontopia. A working installation of Ontopia needs to be present (point to it through the OTM_BASE environment variable in case it cannot be found automatically). Note, though, that's it not required to run the application server, only the JAR files are needed.
